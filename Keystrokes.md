MASHENSTEIN KEYSTROKES
=====================

Gameplay / general controls
- Space, ArrowUp, W: jump
- ArrowDown, S: duck
- X, Shift: ability / power
- ArrowLeft, A: move left
- ArrowRight, D: move right
- Enter, Space: confirm
- Backspace: back
- Escape: escape / back (context-dependent)
- P: pause
- M: mute
- F2: toggle debug view

Dev overlay shortcuts (dev builds)
- `: open/close the dev menu
- [: slow down game speed
- ]: speed up game speed
- \: pause/resume
- .: step one frame while paused
- Tab: take over controls
- Ctrl/Cmd + Shift + P: save a screenshot

When the dev menu is open
- ArrowUp / ArrowDown: move selection
- ArrowLeft / ArrowRight: adjust the selected option
- Enter / Space: activate the selected item
- Backspace / Escape: go back / close the menu
